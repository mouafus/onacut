## ONACUT-BACKEND


### HOW TO SET UP

```
# Put appropriate parameters inside .env.example
make install-deps
```

### HOW TO LAUNCH

#### DEV

```
make run

# To run tests
make test

# formating
make lint
```

#### PROD

Or Using Docker

```
# This will install and run your project on port 5000
make docker-run
```
